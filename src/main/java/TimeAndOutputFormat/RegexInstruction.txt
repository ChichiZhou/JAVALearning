Regex是用来定义 search pattern 的。而这个 search pattern 是为了从 text 中寻找某些字符串的。

Meta 字符指的是将 search pattern 抽象化之后的结果。

Meta Character (Meta 字符)

单个字符
\d  表示 0-9 的数组      如果用 \D 表示不是0-9的任何字符
\w  表示 A-Za-z0-9
\s  表示空格
.   表示任何字符


表示数量的字符
* 表示出现 0 次或者 多次
+ 表示出现 1 次或 多次
? 表示出现 0 次或 1 次
{} 表示出现次数的范围


表示位置的字符

^ 表示初始
$ 表示最后
\b 表示字符边界 用来表示字符的隔离

字符集 []
[] 表示可以匹配里面的任意字符，不用在意数量  如果 - 不是第一个字符，那么就不表示字面上的 - ，而表示从某个字符到某个字符

整体符号 ()
() 里面的字符都将会被看成一个整体来进行匹配



CAPTURE GROUPS
默认情况下，regex 会将整体当做 group 0，然后每个括号里面的字符从 group 1 到 group N




特殊处理：
* 会有一个 greedy match 在里面。如果不想做成 greedy match，则在 * 后面加一个 ?。即  *?
